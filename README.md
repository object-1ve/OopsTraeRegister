# OopsTraeRegister

OopsTraeRegister 是一个自动化注册 Trae 账号的工具脚本集合。它集成了获取临时邮箱、发送验证码、获取验证码邮件以及自动完成注册流程等功能，支持多线程批量并发注册，并将成功注册的账号保存至本地。

## 功能特性

- **自动化注册**：自动完成 Trae 平台的账号注册流程。
- **临时邮箱**：动态生成 `sunix.eu.org` 临时邮箱，避免使用真实邮箱。
- **自动收码**：自动拉取并解析验证码邮件内容，提取 6 位验证码。
- **并发批量注册**：支持单账号注册与多线程批量并发注册。
- **本地持久化**：自动将注册成功的账号和密码信息保存到 `success_accounts.json`。
- **二维码支持**：提供获取并展示二维码的辅助脚本。

## 核心文件结构

- `run_flow.py`: 主流程控制脚本，串联获取邮箱、发码、收码和注册，支持批量并发执行。
- `getCode.py`: 发送注册验证码请求。
- `getmail.py`: 通过临时邮箱服务接口轮询拉取邮件并提取验证码。
- `getQRCode.py`: 扫码相关逻辑，下载并展示二维码。
- `register.py`: 注册逻辑核心参考。
- `success_accounts.json`: 注册成功后的账号数据保存文件。
- `git_logs.md`: 项目版本及修改日志。

## 快速开始

### 依赖安装
确保已安装 Python 3 环境，并安装所需的第三方库：
```bash
pip install requests
```

### 单个注册
直接运行脚本，不传参数会自动生成邮箱并使用默认密码 `qwe123456`：
```bash
python run_flow.py
```

指定邮箱和密码：
```bash
python run_flow.py [指定邮箱] [指定密码]
```

### 批量并发注册
使用 `-n` (总数量) 和 `-t` (并发线程数) 参数进行批量注册：
```bash
# 示例：使用 5 个并发线程，总共注册 10 个账号
python run_flow.py -n 10 -t 5
```

## 注意事项

- 运行过程中由于网络波动或目标服务器风控，可能会出现超时或请求失败，脚本内置了异常捕获和重试降级机制。
- 注册成功的账号会自动写入 `success_accounts.json`，且已添加线程锁避免并发写入冲突。
- 仅供学习和自动化测试参考，请合理使用。
